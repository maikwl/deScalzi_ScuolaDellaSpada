Welcome

This is a lightweight wiki with a book-like structure. Each chapter page presents two aligned columns: the original Italian text and its translation, broken into matching boxes.

How to use

  1. Duplicate a chapter file (e.g., chapters/chapter-01.html) for each new chapter.
  2. Inside each chapter, add one .pair per segment. The left box is the Italian original, the right box is the translation.
  3. Use the "Toggle translation" button to hide/show the translation column during editing or review.
